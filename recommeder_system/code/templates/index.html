Movie Search Engine/Recommerder

About

The goal of Movie Search Engine is meant to be a complementary of Rotten tomato. We want to build a more succinct search engine/recommendation system about movies. Specifically,Users get movies recommendation if they have a preference genre or they can type whatever they want and we will provide some related movies.The query has to be at least somewhat reasonable or we will not be able to provide any useful information. The output two scores; one is fully based on the similarity of the query, another is to combine similarity with the rating of the movie. We also provide some visulization for different genre of movies because we know a picture is worth a thousand words! If you want to know more information, which we do not have due to our building philosophy, we provide you the link to the Rotten tomato with the specific movie which has much more information .
